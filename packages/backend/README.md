# SocialBook | Backend

ExpressJS

## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`MONGO_URL`

`ACCESS_TOKEN_SECRET`

`PORT`

`APP_URL`

#### Cloudinary Storage

`CLOUDINARY_CLOUD_NAME`

`CLOUDINARY_API_KEY`

`CLOUDINARY_API_SECRET`

#### Google SMTP 

`GOOGLE_SMTP_HOST`

`GOOGLE_SMTP_PORT`

`GOOGLE_SMTP_USER`

`GOOGLE_SMTP_PASSWORD`
